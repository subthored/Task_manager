## About:

Simple task tracker working on Laravel

## Requirements
> - PHP 8.2
> 
> - Composer
> 
> - Node.js and NPM
> 
> - Sqlite for local database
> 
> - Make
 
## Installation

> - $git clone https://github.com/subthored/php-project-57.git
> 
> - $cd php-project-57
> 
> - $make install
> 
> - $make database
> 
> - $make migrate
> 
> - $make seed
> 
> - $make start
